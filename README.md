## AI Todo App Projek
Sometime we forgot about our task that effect our future. I have an idea develop an todo web app that have feature like nofication and ai chat bot for helping us choose a task wisely.

## Features
- Authocation System
- Display user task list
- Add new task to the list
- Remove task from the list
- Edit task from the list
- Saving data on database
- AI chatbot

## Requirments
- Front-end : React.js
- Back-end  : Node.js + Express
- Database  : PostgreSQL
- Auth Sys  : JWT
- AI API    : Google Gemini (free version)
